  • 📖 Overview

    🔹 A utility-driven computer vision project designed to automate the tedious task of cropping faces from large datasets. Using Python and OpenCV, the tool scans images, identifies human faces with high precision, and automatically crops and saves them as individual files. This is highly useful for preparing datasets for machine learning training, profile picture standardization, and bulk graphic design tasks.
     

    💻 Tech stack & More

    ✅ Python Programming
    ✅ OpenCV (Open Computer Vision)
    ✅ Automated Face Detection
    ✅ Bulk Image Processing
    ✅ Customizable Crop Ratios
    ✅ Intelligent Dataset Preparation
