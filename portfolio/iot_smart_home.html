  • 📖 Overview

    🔹 This project is a comprehensive smart home management dashboard built with the Laravel framework. It provides a centralized web interface for controlling and monitoring IoT devices in real-time. Features include device state management, historical data visualization, and secure user authentication, making home automation accessible and intuitive.
     

    💻 Tech stack & More

    ✅ Laravel (PHP Framework)
    ✅ Real-time Event Broadcasting
    ✅ Blade Templating Engine
    ✅ MySQL Database ORM
    ✅ REST API Integration
    ✅ Responsive Control Panel
