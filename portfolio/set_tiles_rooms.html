  • 📖 Overview

    🔹 A practical computer vision application that helps users visualize how different floor tiles would look in their own rooms. By uploading a photo of a room, the system detects the floor area using image segmentation and perspective transformation, then overlays high-quality tile textures onto the floor surface. This allows for realistic interaction and fast decision-making for interior design and renovation projects.
     

    💻 Tech stack & More

    ✅ Computer Vision / Image Processing
    ✅ Perspective Transformation
    ✅ Image Segmentation
    ✅ Texture Mapping & Overlay
    ✅ Interior Design Visualization
