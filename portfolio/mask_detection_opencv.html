  • 📖 Overview

    🔹 A computer vision-based safety enforcement tool designed to detect the presence of face masks in real-time. Powering by OpenCV and Convolutional Neural Networks (CNN), the system analyzes video feeds to identify individuals and categorize them as "Wearing Mask" or "Not Wearing Mask." This project aims to assist in public health compliance and safety monitoring in crowded environments.
     

    💻 Tech stack & More

    ✅ Python / OpenCV
    ✅ CNN (Convolutional Neural Networks)
    ✅ Real-time Video Stream Analytics
    ✅ Custom Classification Model
    ✅ Automated Alerts & Logging
    ✅ High-speed Performance
